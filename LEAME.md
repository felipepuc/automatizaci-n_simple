# Automatización simple - Chatbot v1.0.0

Chatbot basado en reglas para consultar:

- Clima o temperatura actual en una ciudad.
- Precio de la acción de una empresa.

## Ejecutar

```bash
python main.py
```

## Instalar dependencias

```bash
pip install -r requisitos.txt
```

## Ejemplos

```text
--> precio de la acción de Microsoft
>>> El precio actual de la acción de Microsoft es $...

--> clima en Oaxaca
>>> La temperatura actual en Oaxaca es +...
```
